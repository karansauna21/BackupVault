# Supported Versions & Platform Matrices

This page outlines the platforms, architectures, and framework versions supported by BackupVault.

---

## 1. Operating System Compatibility

| OS Profile | Target Version | Architecture | Status |
| ---------- | -------------- | ------------ | ------ |
| Windows    | 10 (1607+) / 11| x86_64, ARM64| Native |
| Android    | 8.0 (API 26+)  | arm64-v8a    | Native |

---

## 2. Flutter / Dart SDK Targets

BackupVault is developed and tested against the following core software versions:

- **Flutter SDK**: `3.22.x` (Stable Channel)
- **Dart SDK**: `3.4.x` (Stable Channel)
- **C++ Compiler**: MSVC 2022 (v143 or newer)
- **Android Gradle Plugin (AGP)**: `8.3.x`
- **Gradle**: `8.4`
